Experiment X - Alternate Worlds


Description

Procedurally generates a tile-based world using a seeded random function and ASCII representation. You can reseed to explore alternate world variations.

Technical

This sketch uses p5.js to build a tile-based ASCII environment. Tiles are drawn from a pixel art tileset and updated based on the seed.

Reflection

This project helped me understand how randomness, noise seeds, and procedural generation work together in creative coding.
