Description

This project explores generative art through an evolutionary algorithm that attempts to visually approximate a target image using randomly generated rectangles. Inspired by biological evolution, the system continuously mutates a pool of rectangles to improve its resemblance to a chosen image. Users can select from several well-known reference artworks and adjust the mutation rate in real time. The goal is to simulate how simple rules and variation can converge toward recognizable forms over time.

Technical

The project is implemented in JavaScript using the p5.js creative coding library. It begins with a randomly generated set of colored rectangles layered over a uniform background. In each animation frame, the system mutates this design slightly and evaluates how closely it matches the pixel values of the inspiration image. If the new version performs better according to a pixel-by-pixel similarity metric, it replaces the previous best design. The original image is displayed alongside the evolving design, and the best result is archived using HTML canvas snapshots. The mutation rate is adjustable via a slider, allowing users to influence convergence speed.

Reflection

Working on this project deepened my understanding of how simple evolutionary algorithms can be applied outside typical computer science topics like algorithms or data science. One challenge I encountered was ensuring performance remained fast while comparing large pixel arrays every frame, especially with high-resolution images. Resizing inspiration images and optimizing canvas size helped resolve this issue. I also gained experience integrating p5.js with dynamic HTML controls, which improved the interactivity of the system.I found that working on this project felt pretty rewarding as the results look super cool.
