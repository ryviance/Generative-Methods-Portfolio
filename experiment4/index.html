Experiment 4 - Procedural Tile World

Description

This experiment explores an infinite procedural tile world using p5.js and Perlin noise, complete with interactivity and scrolling.

Technical

The world is generated on-the-fly using Perlin noise, with arrow-key camera controls and seed-based regeneration. Clicking on tiles marks them, and the canvas resizes dynamically.

Reflection

This project challenged me to manage canvas resizing, camera logic, and procedural generation in one cohesive system. It was rewarding to see it all come together.
